# 材料
- 卵 : 10個
- 長ネギ : 1/2本
- 白ゴマ : 大さじ1
- ニンニクチューブ : 2cm
- しょうがチューブ : 2cm
- 豆板醤 : 小さじ1/2
- ごま油 : 大さじ1
- みりん : 100ml
- 醤油 : 80ml
- 水 : 200ml
- めんつゆ : 大さじ1
- 砂糖 : 大さじ1
- お湯 : 2リットル
- お酢 : 大さじ2

# 手順
- ごま油、生姜、にんにく、豆板醤、長ネギのみじん切りを入れて、にんにくの匂いがするまで炒める
- みりん、醤油、水、めんつゆ、砂糖を加えて軽く温める
- ごまを入れてつけダレの完成
- ゆで卵をつけダレに1時間つける

# 反省点
- ゆで卵は6分半だと短かった。8分半ぐらいは必要かと思う。
